# 🎨 새로운 대시보드 빠른 시작

## ✨ 완전히 새로워진 UI/UX!

### 🚀 즉시 시작

```bash
# 1. 대시보드 시작
bash scripts/start_all.sh

# 2. 브라우저에서 접속
http://211.180.253.250:7020
```

## 📊 새로운 기능

### 메인 대시보드 (/)
- **6단계 파이프라인 시각화**
  - 데이터 수집 → 전처리 → 품질 검증 → V100 준비 → 학습 → 결과
- 실시간 진행 상태
- 단계별 클릭 네비게이션

### 페이지 구조

```
/                    # 메인 - 전체 파이프라인 오버뷰
├── /data            # 📦 데이터 관리 (199,994 샘플)
├── /preprocessing   # ⚙️ 전처리 상태 및 결과
├── /quality         # 🔍 품질 검증 (93.7% 점수)
├── /setup           # 🔧 V100 환경 설정
├── /training        # 🚀 실시간 학습 모니터링 ⭐
└── /results         # 📊 학습 결과 및 평가
```

## 🎯 주요 화면

### 1. 메인 대시보드
- 4개 메트릭 카드 (그라데이션 효과)
- 파이프라인 타임라인 (6단계)
- 시스템 정보

### 2. 학습 모니터링 ⭐
- 학습 단계 진행상황
- GPU 사용률 실시간 표시 (V100 × 2)
- Epoch 및 Loss 추적
- V100 최적화 설정
- 실시간 로그

### 3. 데이터 관리
- 전체 데이터 현황
- Train/Val/Test 분할
- 해상도 분포 차트
- 카테고리 통계

## 📝 실행 예제

### 1. 대시보드 확인

### 3. 학습 시작
- /training 페이지 이동
- 명령어 확인 및 실행
- 실시간 모니터링
