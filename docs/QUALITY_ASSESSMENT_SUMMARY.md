## 🎯 우선순위별 개선 로드맵

### 🔴 Week 1 (Critical - 즉시 실행)

**1. CLIP Score 측정** (3일, GPU 48시간)
```bash
python scripts/calculate_clip_score_batch.py \
  --input_csv all_train.csv \
  --sample_size 5000 \
  --model ViT-B/32 \
  --output clip_scores.csv
```
- 예산: $200~$400
- 목표: 현재 CLIP Score 파악
- 예상: 0.25~0.30 (경계선)

**2. FVD 측정** (4일, GPU 32시간)
```bash
# Reference dataset 준비
wget https://kinetics.download/.../kinetics700.tar.gz

# FVD 계산
python scripts/calculate_fvd_batch.py \
  --dataset_csv all_train.csv \
  --reference_dir ./kinetics_reference \
  --sample_size 2000 \
  --output fvd_score.json
```
- 예산: $300~$600
- 목표: 현재 FVD 파악
- 예상: 800~1500 (불확실)

### 🟠 Week 2-3 (High Priority)

**3. 해상도 일괄 변환** (3일)
```bash
python scripts/batch_resize.py \
  --input_csv all_train.csv \
  --target_resolution 1280 720 \
  --output_dir ./resized \
  --workers 16
```
- 영향: 77.8% (155,505개) 해결
- 예산: 스토리지 200GB ($10)

  --output_dir ./trimmed
```
- 영향: 14.9% (29,800개) → 2% 미만
- 예산: GPU 16시간 ($80)

  --input_csv all_train.csv \
  --fix_duplicates \
  --fix_media_type \
  --make_concrete \
  --output_csv all_train_improved.csv
```
- 영향: CLIP Score +0.05~0.08 예상
